# Khata Book 📒

Credit/Debit manager with native SQLite, bill creation, and Google Drive sync.

## Build APK (no Android Studio needed)

1. Push this repo to GitHub
2. Go to **Actions** tab
3. Click **Build Khata Book APK** → **Run workflow**
4. Wait ~10 minutes
5. Download APK from **Artifacts** section

## Import your existing data

Open app → Settings → Restore from .db File → select your creditdebit DB

## DB File location on device

`/data/data/com.khatabook.app/databases/khatabook.db`

Fully compatible with Ankit Saraf Credit/Debit app format.
